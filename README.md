# medFactoraalpha

QuantaAlpha-like / LLM-guided symbolic medical factor discovery.

## Research Idea

Use LLMs or agents as medical knowledge priors to propose:

- clinically meaningful variables,
- time windows,
- symbolic operators,
- formula seeds,
- search-space constraints,
- feature/factor explanations.

Then use data-driven evaluation and evolutionary/RL-style search to discover interpretable medical factor libraries for clinical prognostic prediction.
